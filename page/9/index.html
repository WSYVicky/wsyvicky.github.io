2018-01-24

利用Hexo搭建自己的个人博客（MacBook Pro）

(一)环境安装

  1. npm的安装
  2. hexo的安装

    $ npm install hexo-cli -g

（二）本地博客搭建

  1. 本地初始化一个名为“blog”的文件夹

    $ hexo init blog
  2. 进入blog文件夹

    $ cd blog
  3. 在文件夹中安装npm

    $ npm install
  4. 启动hexo sever

    $ hexo server

（三）将本地blog部署到GitHub上

  1. 在GitHub上新建一个profile
  2. 域名填写要根据GitHub账户名来定。
  3. 配置本地ssh。在终端输入：

    $ git config --global user.name "WSYVicky"
    $ git config --global user.mail "1248965945@qq.com"
  4. 生成本地公钥和秘钥

    $ ssh-keygen -t rsa -C "1248965945@qq.com"

（四）修改本地.ssh/文件

  1. 用编辑器打开.ssh/文件下的id_rsa.pub文件

  3. 测试ssh能不能打通这个账号,在终端输入：

    ssh -T git@github.com
    显示successfully就是成功了！

    $ hexo deploy

（6）修改主题

  1. 在hexo的官网上有很多主题，我这里用了一个icarus的主题，然后打开他的GitHub，按照步骤去下载。
  2. 修改blog下的_config.yml文件,将theme改成下载好的主题名字。
    theme: icarus
  3. 将修改好的_config.yml文件复制到blog/themes/icarus目录下
    $ cp _config.yml.example _config.yml
  4. $ hexo s去http://localhost:4000/看是否成功。
  5. $ hexo clean && hexo g -d清除本地垃圾，并且完成部署！
